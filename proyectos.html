Tienda Online Moda

E-commerce

Descripción del Proyecto

Diseño y desarrollo de una tienda online completa para una marca de moda. El sitio incluye catálogo de productos, sistema de filtrado, carrito de compras y pasarela de pago segura.

El diseño se centró en destacar las imágenes de los productos y ofrecer una experiencia de usuario fluida y atractiva, con especial atención a la versión móvil.

Tecnologías Utilizadas

  • HTML5
  • CSS3
  • JavaScript
  • PHP
  • MySQL
  • API de Pagos

Características Principales

  • Diseño responsivo
  • Catálogo de productos
  • Sistema de filtrado avanzado
  • Carrito de compras
  • Pasarela de pago segura
  • Panel de administración
  • Gestión de inventario
